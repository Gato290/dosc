VelarisUI punya built-in Config System untuk menyimpan dan memload nilai element (Toggle, Slider, Dropdown, Input) secara otomatis ke file JSON.

💡 Untuk elemen yang ingin disimpan ke config, tambahkan property Flag = "nama_key" pada config elemen tersebut.

Setup Config

local Window = VelarisUI:Window({ Title = "My Hub", Config = { ConfigFolder = "MyHub/Config/", -- folder tempat file JSON disimpan AutoSaveFile = "Default", -- nama file .json (tanpa ekstensi) AutoSave = true, -- simpan otomatis setiap perubahan AutoLoad = true, -- load otomatis saat window dibuat }, })

Flag pada Elemen

-- Toggle dengan Flag Sec:AddToggle({ Title = "Auto Farm", Flag = "auto_farm", -- key di config file Default = false, Callback = function(v) end }) -- Slider dengan Flag Sec:AddSlider({ Title = "Speed", Flag = "walk_speed", Min = 1, Max = 100, Default = 16, Callback = function(v) end })

Manual Save / Load

-- Simpan ke file bernama "Default.json" Window:SaveConfig("Default") -- Load dari file Window:LoadConfig("Default") -- Hapus file config Window:DeleteConfig("Default") -- List semua file config yang ada local list = Window:ListConfigs() for _, name in ipairs(list) do print(name) end -- Reset semua elemen ke default & hapus file Window:ResetConfig("Default")

local Window = VelarisUI:Window({ Title = "My Hub", DiscordSet = { Enable = true, Title = "My Server", -- nama ditampilkan di atas tombol Link = "discord.gg/xxxxx", -- link yang di-copy saat diklik Icon = "lucide:message-circle", -- opsional }, })

Tambahkan video sebagai background window. Mendukung Roblox Asset ID maupun URL file video eksternal.

local Window = VelarisUI:Window({ Title = "My Hub", BackgroundVideo = "rbxassetid://VIDEO_ID", BackgroundVideoOverlay = 0.4, -- transparansi overlay gelap (0-1) })

Gunakan property Tabbox untuk membagi section menjadi dua kolom berdampingan. Nilai "Left" dan "Right" akan berpasangan otomatis dalam satu baris di dalam Tab yang sama.

💡 Section dengan Tabbox = "Left" dan Tabbox = "Right" akan tampil berdampingan secara otomatis. Keduanya harus dibuat secara berurutan di dalam Tab yang sama.

Dengan Icon

local BoxLeft = Tabs.Main:AddSection({ Title = "Auto Farm", Icon = "lucide:cpu", Tabbox = "Left", }) local BoxRight = Tabs.Main:AddSection({ Title = "Settings", Icon = "lucide:settings", Tabbox = "Right", })

Tanpa Icon (Text Saja)

local BoxLeft = Tabs.Main:AddSection({ Title = "Auto Farm", Tabbox = "Left", }) local BoxRight = Tabs.Main:AddSection({ Title = "Settings", Tabbox = "Right", })

Contoh Lengkap dengan Elemen

local BoxLeft = Tabs.Main:AddSection({ Title = "Auto Farm", Icon = "lucide:cpu", Open = true, Tabbox = "Left", }) local BoxRight = Tabs.Main:AddSection({ Title = "Settings", Icon = "lucide:settings", Open = true, Tabbox = "Right", }) -- Tambahkan elemen seperti biasa ke masing-masing box BoxLeft:AddToggle({ Title = "Auto Collect", Default = false, Callback = function(v) end }) BoxRight:AddSlider({ Title = "Speed", Min = 1, Max = 100, Default = 16, Callback = function(v) end })

PropertyTypeDefaultDescription

⚠️ Catatan: Section Tabbox = "Left" harus selalu dideklarasikan sebelum Tabbox = "Right" agar layout terbentuk dengan benar.
🔒 Locked pada Section akan menampilkan overlay kunci di atas seluruh elemen di dalam section tersebut.

Notification
Tampilkan toast notification ke user
ℹ️ Ada dua cara menampilkan notifikasi: fungsi global Notify() / Nt(), dan method Window:Notify() yang lebih modern dengan fitur lebih lengkap.
Global Notify
Window:Notify
MakeNotify

Basic

Notify("Ini isi pesan")

Lengkap

Notify( "Script berhasil diload!", -- Content 5, -- Delay (detik) "Default", -- Color "My Script", -- Title "Success" -- Description )

Method Window:Notify() adalah sistem notifikasi terbaru dengan dukungan progress bar otomatis, tombol aksi, dan layout yang lebih clean.

Basic

Window:Notify({ Title = "Berhasil", Content = "Script telah diload dengan sukses!", Duration = 5, })

Dengan Tombol

Window:Notify({ Title = "Update Tersedia", Content = "Versi baru v2.0 sudah tersedia!", Duration = 0, -- 0 = tidak auto-close Buttons = { { Name = "Update", Primary = true, Callback = function() print("Update clicked") end }, { Name = "Nanti", Callback = function() end }, }, })

Properties

PropertyTypeDefaultDescription
Titlestring"Notification"Judul notifikasi
Contentstring""Isi pesan notifikasi
Durationnumber5Durasi tampil dalam detik. 0 = tidak auto-close
Buttonstable{}List tombol aksi. Setiap tombol punya Name, Callback, dan Primary (boolean)
💡 Saat ada Buttons, progress bar tidak ditampilkan dan notifikasi tidak auto-close kecuali Duration > 0.
local Notif = Chloex:MakeNotify({ Title = "My Script", Description = "Success", Content = "Script telah diload dengan sukses!", Color = "Default", Time = 0.5, Delay = 5, Icon = "lucide:check", Buttons = { { Name = "OK", Primary = true, Callback = function() end }, }, }) -- Tutup notifikasi secara manual Notif:Close()

Window
Membuat window utama VelarisUI

Create Window

local Window = VelarisUI:Window({ Title = "Velaris UI", -- Main title di atas window Footer = "By Nexa", -- Footer text di bawah Content = "Content", Color = "Default", -- Tema warna UI Version = 1.0, ["Tab Width"] = 120, -- Lebar section tab Image = "101833678008843", -- Asset ID icon window Configname = "Vilaris_Ui", -- Nama file konfigurasi Uitransparent = 0.15, -- Transparansi UI (0=solid) ShowUser = true, Search = true, Config = { AutoSave = true, -- Auto simpan settings AutoLoad = true, -- Auto load settings }, })

Properties

PropertyTypeDescription
TitlestringJudul utama di atas window.
FooterstringTeks footer di bagian bawah.
ColorstringTema warna UI. Default: "Default".
VersionnumberVersi script yang ditampilkan.
Tab WidthnumberLebar section tab (default: 120).
ImagestringAsset ID untuk icon window.
ConfignamestringNama file untuk menyimpan config.
UitransparentnumberTransparansi UI. 0 = solid, 1 = transparan penuh.
ShowUserbooleanTampilkan nama user.
SearchbooleanAktifkan fitur search.
Config.AutoSavebooleanAuto-simpan settings.
Config.AutoLoadbooleanAuto-load settings saat start.

Section
Container untuk mengelompokkan elemen UI

Section adalah container utama untuk button, toggle, slider, dan elemen lainnya.

Frugal Local
Basic
Full
local Sec = {} Sec.Player = Tabs.Main:AddSection({ Title = "Player", Icon = "lucide:user", Open = true }) Sec.Weapon = Tabs.Main:AddSection({ Title = "Weapons", Icon = "solar:home", Open = false })
Tabs.Main:AddSection({ Title = "Player", Icon = "lucide:user", Open = true }) Tabs.Main:AddSection({ Title = "Weapons", Icon = "solar:home", Open = false })

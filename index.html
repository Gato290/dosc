Window
Membuat window utama VelarisUI

Create Window

local Window = VelarisUI:Window({ Title = "Velaris UI", -- Main title di atas window Footer = "By Nexa", -- Footer text di bawah Content = "Content", Color = "Default", -- Tema warna UI Version = 1.0, ["Tab Width"] = 120, -- Lebar section tab Image = "101833678008843", -- Asset ID icon window Configname = "Vilaris_Ui", -- Nama file konfigurasi Uitransparent = 0.15, -- Transparansi UI (0=solid) ShowUser = true, Search = true, Config = { AutoSave = true, AutoLoad = true, }, })

Badge
Property universal — bisa dipakai di semua elemen
ℹ️ Badge adalah universal property. Dapat ditambahkan ke semua elemen: Button, Toggle, Slider, Dropdown, Input, Panel, Paragraph, dan Keybind.

Badge Types

ValuePreviewPenggunaan
"Bug"BugMenandai fitur yang bermasalah
"New"NewMenandai fitur yang baru ditambahkan
"Warning"WarningMenandai fitur yang perlu perhatian
"Fixed"FixedMenandai fitur yang sudah diperbaiki

Penggunaan di Semua Elemen

Cukup tambahkan Badge = "..." ke property elemen manapun.

Button

Sec:AddButton({ Title = "New Feature", Badge = "New", Callback = function() end })

Toggle

Sec:AddToggle({ Title = "Buggy Toggle", Badge = "Bug", Default = false, Callback = function(v) end })

Slider

Sec:AddSlider({ Title = "Speed", Badge = "Fixed", Min = 0, Max = 100, Default = 50, Callback = function(v) end })

Dropdown

Sec:AddDropdown({ Title = "Select Mode", Badge = "Warning", Options = { "A", "B", "C" }, Callback = function(v) end })

Input

Sec:AddInput({ Title = "Username", Badge = "New", Default = "", Callback = function(v) end })

Panel

Sec:AddPanel({ Title = "Action Panel", Badge = "Bug", ButtonText = "Execute", ButtonCallback = function(v) end })

Paragraph

Sec:AddParagraph({ Title = "Changelog", Content = "Fixed crash issue on load.", Badge = "Fixed", })

Keybind

Sec:AddKeybind({ Title = "Toggle ESP", Badge = "Warning", Value = "RightShift", Callback = function(key) end })
Locked
Property universal — disable elemen di semua tipe
ℹ️ Locked adalah universal property. Dapat dipakai di semua elemen untuk menonaktifkan interaksi user. Bisa di-set saat deklarasi maupun dinamis dengan :SetLocked(state).

Cara Pakai

Tambahkan Locked = true pada property elemen manapun. Bisa juga dikombinasikan dengan Badge.

Button

Sec:AddButton({ Title = "Locked Button", Locked = true, Callback = function() end })

Toggle

Sec:AddToggle({ Title = "Locked Toggle", Locked = true, Default = false, Callback = function(v) end })

Slider

Sec:AddSlider({ Title = "Locked Slider", Locked = true, Min = 0, Max = 100, Default = 50, Callback = function(v) end })

Dropdown

Sec:AddDropdown({ Title = "Locked Dropdown", Locked = true, Options = { "A", "B", "C" }, Callback = function(v) end })

Input

Sec:AddInput({ Title = "Locked Input", Locked = true, Default = "", Callback = function(v) end })

Panel

Sec:AddPanel({ Title = "Locked Panel", Locked = true, ButtonText = "Action", ButtonCallback = function(v) end })

Paragraph

Sec:AddParagraph({ Title = "Locked Info", Content = "This element is locked.", Locked = true, })

Keybind

Sec:AddKeybind({ Title = "Locked Keybind", Locked = true, Value = "RightShift", Callback = function(key) end })

Dynamic Lock — SetLocked

Semua elemen mendukung method :SetLocked(state) dan :GetLocked() untuk mengunci/membuka secara dinamis.

-- Contoh pada Toggle (berlaku sama untuk semua elemen) local MyToggle = Sec:AddToggle({ Title = "Dynamic Lock", Default = false, Callback = function(v) end }) MyToggle:SetLocked(true) -- kunci elemen MyToggle:SetLocked(false) -- buka elemen print(MyToggle:GetLocked()) -- true / false

Kombinasi Badge + Locked

Kedua property bisa dipakai bersamaan pada elemen apapun.

Sec:AddButton({ Title = "Coming Soon", Badge = "New", Locked = true, Callback = function() end }) Sec:AddToggle({ Title = "Broken Feature", Badge = "Bug", Locked = true, Default = false, Callback = function(v) end })

Gunakan property Locked = true untuk menonaktifkan button. Berlaku untuk Button V1 maupun V2.

-- Button V1 Locked Sec.Button:AddButton({ Title = "Locked", Locked = true, Callback = function() print("This button is locked") end }) -- Button V2 Locked (dengan Icon) Sec.Button:AddButton({ Title = "Button V2 Locked", Locked = true, Version = "V2", Icon = "rbxassetid://79715859717613", Callback = function() print("This button is locked") end })

Gunakan :SetLocked(state) untuk mengunci/membuka button secara dinamis:

local MyBtn = Sec.Button:AddButton({ Title = "Dynamic Lock", Callback = function() end }) MyBtn:SetLocked(true) -- kunci MyBtn:SetLocked(false) -- buka print(MyBtn:GetLocked()) -- cek status
